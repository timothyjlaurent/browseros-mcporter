# Updating the You.com agent list

Use this when the user asks to refresh available custom agents.

## Procedure
1. Launch BrowserOS (via mcporter) and navigate to https://you.com.
2. Open the agent selector dropdown.
3. Capture all agent names (scroll for full list).
4. Replace the list in `references/agents.md` with the new list.
5. Confirm the update in your reply.
